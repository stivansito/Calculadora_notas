# Calculadora de Notas Bonita

Mini app web para llevar asignaturas con:

- pesos por parte
- nota minima global para aprobar
- partes que deben aprobarse
- partes con minimo propio, como un 4
- calculo automatico de lo que te conviene sacar en lo pendiente

## Como abrirla

La forma mas facil es hacer doble clic en:

- `D:\Métodos_Samu\calculadora_notas_bonita\abrir_calculadora.bat`

Tambien puedes abrir directamente:

- `D:\Métodos_Samu\calculadora_notas_bonita\index.html`

## Como usarla

1. Cambia el nombre de la asignatura.
2. Pon la nota minima para aprobar, normalmente 5.
3. Anade las partes de evaluacion con su peso.
4. En cada parte decide si:
   - no tiene minimo
   - debe aprobarse
   - tiene minimo propio
5. Ve rellenando las notas que ya tengas.
6. Mira la caja de estado y la seccion de objetivos sugeridos para ver que te falta.

## Guardado

La app guarda automaticamente en el navegador. Si quieres una copia, usa el boton `Exportar`.
